 # Base Guess Game

A tiny guessing game smart contract on Base.

Players send a small ETH fee and guess the secret number.

# Network

Base Mainnet  

Chain ID: 8453  

Gas token: ETH on Base

## Contract

`contracts/GuessGame.sol`

## How to deploy

Use Remix:

1. Compile contract

2. Connect MetaMask

3. Select Base Mainnet

4. Deploy with Injected Provider
